# README

## Описание
Данный Python скрипт использует библиотеку `telebot` для создания Telegram-бота, который управляет регистрацией пользователей и просмотром их профилей. Бот взаимодействует с пользователями в чате, позволяя им зарегистрироваться, указав имя и возраст, а затем просматривать информацию своего профиля.

## Функционал
- **Процесс Регистрации**: Пользователи могут зарегистрироваться, указав свое имя и возраст через серию чатовых взаимодействий.
- **Просмотр Профиля**: Зарегистрированные пользователи могут просматривать информацию своего профиля, включая ID, имя и возраст.
- **Взаимодействие с Базой Данных**: Бот хранит данные регистрации пользователей в базе данных SQLite с именем `database_tg.db`.

## Требования
- Python 3.x
- Библиотека `telebot`
- Библиотека `sqlite3`
- Программа `SQLiteStudio`

## Установка
1. Клонируйте репозиторий или загрузите скрипт.
2. Установите необходимые библиотеки с помощью pip:
   
   pip install pyTelegramBotAPI
   
3. Запустите скрипт с помощью Python:
   
   python ваш_скрипт.py
   

## Настройка
- Замените `'YOUR_BOT_TOKEN'` в скрипте на фактический токен вашего Telegram-бота.
- Убедитесь, что в том же каталоге, что и скрипт, есть файл базы данных SQLite с именем `database_tg.db`.

## Использование
1. Запустите бота, запустив скрипт.
2. Взаимодействуйте с ботом в Telegram:
   - Новых пользователей попросят зарегистрироваться.
   - Зарегистрированные пользователи могут просматривать свой профиль.
